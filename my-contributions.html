Chapter One

Hello, I'm Nitesh.

Every engineer has an origin story. Mine began with curiosity — the itch to open things up and see how they work, then the slightly more dangerous itch to put them back together better. Somewhere between the first "Hello, World" and the first production incident at 2 a.m., that curiosity became a craft.

I care about software that lasts: code that reads like prose, systems that fail loudly and recover quietly, and documentation that respects the next person's time. When I'm not building, I'm usually reading, tinkering with side projects, or contributing back to the open-source tools I rely on every day.
